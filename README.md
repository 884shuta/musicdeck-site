# MusicDeck Site

MusicDeckの公式Webサイトです。

## 公開URL

https://884shuta.github.io/musicdeck-site/

## ページ構成

- `index.html`  
  MusicDeckのトップページ

- `privacy.html`  
  プライバシーポリシー

- `terms.html`  
  利用規約

- `styles.css`  
  サイト全体のデザイン

- `app.js`  
  クリック制御などの軽いJavaScript

## GitHub Pages設定

1. GitHubで `musicdeck-site` リポジトリを作成
2. このファイル一式を `main` ブランチにpush
3. GitHubのリポジトリ画面で `Settings` を開く
4. `Pages` を開く
5. Sourceを `Deploy from a branch` にする
6. Branchを `main`、Folderを `/root` にする
7. Saveする
